亲爱的Nora小姐-【空巢青年】剧本

发表于 2021-12-02 更新于 2025-11-21 本文字数： 2k 阅读时长 ≈ 7 分钟

主角就是Nora，是个害羞的男孩子
新的扮演Nora小姐的人（也就是Nora小姐喜欢的人）就是“配角A”，因为他曾经获得过剪刀，所谓“九张机”

第一幕，屋中对话

门铃响，开门，配角 A 把剪刀放在桌子上

配角 A：“剪刀还你，话说最近就要走了吗？”
主角：“嗯，我想大概明天一早就动身。”
配角 A：“保重，需不需要告知 Nora 小姐。”
主角：“不必了，Nora 小姐的情绪状况不稳定，告诉她了心里指不定有多难受。”
配角 A：“好，我会记得你的，那么有缘再见。”

关门

旁白（打字）：没人知道 Nora 小姐姓甚名谁，长什么样，但她的名头却早在社区里传开了。
凭借着一手如沐春风的文字，Nora 小姐逐渐成为了漂流瓶活动最为热门的“知心姐姐”人选。
她的回信抚慰了无数打工人寂寞的内心。十天一度的定期回信，早已成为租户们心照不宣的
默契。

第二幕，屋中回忆【初见 Nora】

打开橱柜，里面有一些药品

主角：“一眨眼竟然已经这么久了啊，总觉得像是昨天才刚搬进来的一样。”
主角：“不过，那时的我…”
切入回忆，镜头：抑郁诊断证明，离职信，药物，酒杯，面具，
旁白（打字）：“沐浴在宁静的黑夜里，仿佛想起以前的那些想起，把头潜到黑夜里，试试看
能不能把眼睛睁开”
主角（独白）：
“一切的起点都要从 Nora 说起，那一天我只是好巧不巧地看到了社区漂流瓶活动，这是全程
匿名的，不会有人知道你是谁。”
“我没报任何回信的想法，草率地写下了第一封来信。”
来信：（打字）
“Nora 您好，我发现我在持续地忘记事情。昨天发生了些什么呢？我打算做些什么呢？大部
分的想法和愿望都不见了。一种自己将要消失的理想逐渐弥漫，然而似乎不曾存在的东西要
怎么消失呢？于是也只是这样隐隐念想着。然而时间实在比我跑得快太多，这又是一周，这
又是一月，这又是一年了。
Nora 会害怕时间吗，会厌恶人们眼中所谓崭新的生活方式吗。”
“没想到她迅速简洁地回了我。”
回信：（打字）
“谢谢你这么信任我，或许只有时间能给你答案，我把魔法灌入了这些活动当中，行动起来
吧，去追寻那些启示录一般的时间，和久经搁置却依旧熟悉的快照。”
“我就在他们中间，要加油啊。”
主角（独白）：
“我是第一次感受到如此真切的共鸣，或许我们会有一丝不期而遇的机会呢（也是第一次催
生了要认识一只下蛋母鸡的想法。）”
“那么，就出去试试吧。”

第三幕，回忆场景切换【屋顶花园】

来信：（打字）
“Nora 您好，最近想试试在屋顶花园种点东西，请问我需要注意点什么呢，会不会被别人当
成杂草呢。”
回信：（打字）
“眼中没有梁木的人，自然会期待土地赠予我们的一切可能。”
主角（独白）：
屋顶花园是我向外界迈出的第一步，Nora 说得对，只要眼中没有梁木，自然不会介意开不
开花、结不结果。
场景：【屋顶花园-种菜讨论】
夜间天台摆设：君子兰、昙花
配角 B：“挺巧的，你一来它就开花了，真好啊”
主角：“是昙花吗，你的手真巧。”
配角 B：“别这么说，随处可见的的品种罢了”
配角 B：“让我也看看你种了什么，挺香的——啊，这是君子兰吗。”
主角：“也是一种随处可见的品种，君子兰是寂寞的花。”
配角 B：“所以你种了个寂寞是吗。”
主角：“或许吧，好在它现在至少能跟你的花在一起香。”
主角（独白）：
“昙花是须臾之花，我跟他的交情会不会也像昙花一样仅在须臾之间呢。”
“不管怎么说，希望我们的花都能健康成长。”

第四幕，回忆场景切换【公共厨房】

来信：（打字）
“Nora 您好，上次种花时被搭讪了，好开心。你的魔法果然有效。听说这里还有一个公共厨
房，我要怎么参与定期的烹饪活动呢。”
回信：（打字）
“面对油锅要拿出表白的勇气，收拾碗筷要拿出初恋的踊跃。”
主角（独白）：
公共厨房是我的第二关挑战，相信我能够拿出足够的勇气与踊跃。
场景：【公共厨房-受伤】
主角：“啊，真倒霉。”
配角 C：“咋了咋了，被食材报复了？”
主角：“没事，一不小心被割破皮了。”
配角 C：“唉，先用这个吧，我听广告上说这个止血效果好。”
主角：“…谢谢您，真是太不小心了。”
配角 C：“没处理完的就放着我来吧，以后小心点就好，这些有多的创口贴拿着吧。”
主角：“真是太谢谢您了。”
主角（独白）：
“虽然我在油锅之前就败下阵来，最后也没能吃上自己做的饭。但炽烈的关怀却让我闻到了，
那些除了饭菜之外的香味。”

第五幕，回忆场景切换【桌游】

来信：（打字）
“Nora 您好，托您的福几个月下来我交到了不少邻居朋友。最近想跟他们打一些桌游，请问
我该怎么申请这边的桌游呢”
回信：（打字）
“除了事先向物业管理处报备之外，还要确保真的交到了几个真心朋友。”
主角（独白）：
虽然我无法保证交到的都是真心朋友，但我至少会付出我的诚意与真心。
场景：【桌游-真心话大冒险】
主角：“好的，目前轮到配角 D 接受惩罚，你的选择是真心话还是大冒险。”
配角 D：“真心话。”
配角 A：“你能接受人是由猴子变来的，我们当中每个人都流着源远流长的猴血吗。”
配角 D：“不能。”
配角 D：“别看了，到你了。我要狠狠地赚回来。”
配角 A：“别得意得太早，大冒险。”
配角 B：“三声猴叫，这里有参考音频。来”
配角 A：【猴叫】
配角 D：“给你个教训，人不应该太纠结于真相的，真相总是带来尴尬。”
配角 D：“怎么又是你，你这也太惨了吧。”
主角：“真心话”
配角 A：“我的起源，我的由来，可能是因为你的存在吗。”
主角：“宇宙是起源于大爆炸的，所以我的起源一定是你。”
主角（独白）：
“交到了这么多的朋友，经过了这么丰富的时光，真是令人欣慰啊。”
“神是会给第二次机会的神，而时空是不会给第二次机会的时空，就让我们珍惜当下的彼此
吧。”

第六幕，回到现实

开门
主角（拖着包慢慢地走）
一群人围上来，配角 A 快步上前

配角 A：“你不让我联系 Nora，我就没跟她说。”
主角：“所以你就所有人都说了是吗。”
配角 A：“准确地讲是所有人都到了。”
配角 B：“我会想你的，屋顶花园为你长留”
配角 C：“记得有空再来这边做点热菜。”
配角 D：“下次再来玩牌会让着你一点的，记得常来啊。”
主角:“谢谢你们，再见啦。”
众人齐声：“再见了，亲爱的 Nora 小姐。”
