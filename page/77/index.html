又又又做了奇怪的梦

发表于 2020-05-29 更新于 2025-11-26 本文字数： 849 阅读时长 ≈ 3 分钟

我发现白天睡觉做梦真就成了定律！并且这些梦的场景也越来越光怪陆离了！
今天这个梦比较单一，主要是在高考发生期间

我好像是爱恋上了一个以前一班的同学，黄星（虽然我貌似好像真的暗恋过她，她似乎也暗恋过我？）然后她要高考，我为了陪着她所以我也要参加高考，并且所剩时间仅剩3天？（并且我觉得还非考不可，然后我打开习题集一看，全部都是初中的几何证明题之类的，我当时觉得自己不会做了，但我突然想到我自己已经上了大学，但是难道上了大学就可以不参加高考了吗？我当时没想清楚，于是就很绝望，然后我妈妈就说你可以不参加（我怎么觉得这有点像逃避兵役的样子啊？））然后我就没有参加，然后我在高考之后把黄星的一些文具啊什么的带走了，然后不知道怎么着，我觉得应该要还给黄星
于是我就开始跑啊跑，我不知道为什么我还带着一床棉被，这床棉被很舒服，就像我现在盖的这床一样，然后我坐着公交车跑到轮渡，然后坐着公交车离开了小岛，然后我抱着棉被发现岛外的环境非常恶劣，到处都是脏污，有一个清洁工女人露张着肮脏的阴道，我看到了直犯恶心，然后我就丢下了棉被继续跑，我想先找到黄星再说，于是我决定回岛？
然后我就觉得，我现在已经离开了小岛，那么我肯定要从轮渡才能回到小岛不是吗？那我就去问了一下旁边的人，他的右下方我记得是下着滂沱大雨，并且很高，下面是大海（好像我们站在悬崖上？）然后他说现在船很难搭，并且车子也很难搭，我就问那在哪里坐车，然后另一个人就跟我指了下车站的方向，并且告诉我车站离目的地很远，我看到了车站之后，松了口气，突然想到我自己落下了棉被，于是我就回头跑去决定把棉被带回来，然后我在另一头的车站那边突然发现了我妈妈，她还是那一般年轻的样子，她和我拥抱起来，然后我们就回到加了，不知道为什么棉被也回来了
然后我妈妈就跟我说，你把这么多东西交给黄星，已经可以了，我们家的棉被你就可以留下来，我觉得这话在理，但是我文具还没有交还给她，我妈妈就说没问题的，她会接收到的，然后又拿出了一个绿色的笔袋，说你看这个笔袋是陈明骏的，他现在回了趟老家没法拿，你就先用着，我说我不有（自己的笔袋）吗？然后我就拿出了自己的笔袋，是红色的，然后我妈妈就放心了，我也觉得有了这个笔袋可能（考试就稳了？）（考试结束就稳了？）

又做了一些奇怪的梦，但是这个梦也蛮有趣的，于是记录一下
我发现我貌似只有在早上时候睡觉才会做梦，很好玩
今天的梦镜有几个特点，就是它是复合的梦镜，每个梦境有所交织，但并不是统一的一个世界观
